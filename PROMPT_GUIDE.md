# Guia de Prompts — MTRX LIBRARY

> Para a equipe criativa. Sem precisar saber de tecnologia.

---

## O que é um Prompt?

Um prompt é o **comando que você dá para a IA**. É o texto que você digita para ela entender o que você quer criar — seja uma imagem ou uma música.

Quanto mais específico o prompt, melhor o resultado.

---

## Como usar o site

1. Abra o site: https://mtrx-library.vercel.app
2. Navegue pelas categorias ou use a busca
3. Encontrou um prompt interessante? **Clique no texto laranja** — ele é copiado automaticamente
4. Cole na ferramenta de IA que você está usando
5. Combine com outros prompts para resultados melhores

Agora a biblioteca tambem mostra um resumo do banco: total de prompts, tags, combos e status dos dados. Use isso para entender se voce esta vendo a base completa ou um recorte filtrado.

## Prompts de Imagem

### Ângulos de Câmera

São o "ponto de vista" da imagem. Definem de onde a câmera está "olhando".

| Prompt | O que faz | Quando usar |
|---|---|---|
| **Close-up** | Foco total no rosto/detalhe | Para imagens de artistas, retratos emocionais |
| **Wide Shot** | Mostra o ambiente completo | Para mostrar palco, cenário, contexto |
| **Bird's Eye View** | Vista de cima | Para mostrar grandiosidade, multidões |
| **Low Angle** | Câmera de baixo para cima | Para transmitir poder e autoridade |

### Ambientação

Define o clima, a luz e o ambiente da imagem.

| Prompt | O que faz | Quando usar |
|---|---|---|
| **Cinematic Lighting** | Luz dramática de filme | Para qualquer imagem mais séria/impactante |
| **Stage Smoke & Lights** | Fumaça + luzes de show | Para imagens de performance ao vivo |
| **Neon Cyberpunk** | Luzes coloridas, chuva, urbano | Para estética moderna e vibrante |
| **Golden Hour** | Luz dourada do pôr do sol | Para algo mais quente e nostálgico |

### Sujeitos

O que/quem está na imagem.

| Prompt | O que faz | Quando usar |
|---|---|---|
| **Cantor com Microfone** | Artista segurando microfone em performance | Base para qualquer visual musical |
| **DJ Behind Decks** | DJ nos equipamentos com luz de balada | Conteúdo de show/evento |
| **Artista Solo** | Artista sozinho em foco total | Capa, redes sociais, identidade visual |

### Fórmula básica de imagem

```
[SUJEITO] + [ÂNGULO] + [AMBIENTAÇÃO]
```

**Exemplo:**
```
singer holding a microphone, close-up shot, stage smoke and blue lights, cinematic lighting
```

## Prompts de Áudio (Suno)

### Gêneros Musicais

| Prompt | O que é |
|---|---|
| **Sertanejo** | Violão, sanfona, voz melódica brasileira |
| **Sertanejo Universitário** | Versão mais pop/dançante do sertanejo |
| **Funk Brasileiro** | Beat eletrônico pesado, energia urbana |
| **Pagode** | Pandeiro, cavaquinho, samba |
| **Trap BR** | 808 bass, hi-hats rápidos, rap em português |

### Modulação Vocal

| Prompt | O que faz |
|---|---|
| **Voz Grossa** | Voz grave, com autoridade e peso |
| **Voz Suave & Melancólica** | Voz delicada, para baladas |
| **Voz de Narrador** | Voz clara e assertiva, para intros e locuções |
| **Coral / Vozes em Camadas** | Múltiplas vozes sobrepostas, grandioso |

### Mood & Atmosfera

| Prompt | O que transmite |
|---|---|
| **Melancólico & Sad** | Tristeza, reflexão, saudade |
| **Festivo & Animado** | Energia de festa, animação |
| **Romântico & Sensual** | Clima íntimo, amor |

### Fórmula básica de áudio

```
[GÊNERO] + [VOZ] + [MOOD]
```

**Exemplos:**
```
sertanejo style, deep voice, melancholic mood, acoustic guitar
```
```
Brazilian funk, energetic vocals, party atmosphere, heavy bass
```

## Dica rápida

Se você tiver uma ideia mas não souber o prompt exato, abra o site e:

1. Use a **busca** para encontrar palavras relacionadas
2. Veja a **Curadoria do Time** — são os prompts mais usados e eficientes
3. Copie e **adapte** — não precisa usar exatamente como está, combine com suas ideias

## Como a base vai crescer

A biblioteca deve ganhar mais material por contexto de uso:

| Contexto | Exemplos de busca |
|---|---|
| Capa de música | `capa`, `artista`, `editorial`, `studio` |
| Reels | `reels`, `movimento`, `transicao`, `energia` |
| Videoclipe | `videoclipe`, `palco`, `camera`, `performance` |
| Música | `Suno`, `sertanejo`, `funk`, `trap`, `voz` |
| Atendimento | `WhatsApp`, `venda`, `follow-up`, `relacionamento` |

Quando cadastrar novos prompts, pense sempre em tres coisas: **para que serve**, **qual ferramenta usa** e **em qual situacao a equipe vai procurar**.
