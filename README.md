# MTRX LIBRARY — Banco de Prompts Criativo

> Liberdade de criar. Sem bloqueio criativo.

---

## O que é

O **MTRX LIBRARY** é um banco de dados de prompts para geração de imagem e áudio com IA. Funciona como um site aberto — sem login, sem contas — acessível para toda a equipe ao mesmo tempo, como se fosse um PDF interativo e bonito.

O objetivo é simples: **ninguém da equipe fica parado por falta de ideia ou de saber qual prompt usar.**

---

## O problema que resolve

> *"Ele só não concluiu o projeto porque faltou isso. Se ele tivesse o banco de dados, ele poderia iniciar."*

Criadores bloqueiam no meio do projeto quando não sabem qual prompt usar. O MTRX LIBRARY resolve isso dando acesso imediato a comandos prontos, organizados e testados.

---

## Como funciona

### Para usuários (toda a equipe)
- Acesse o arquivo `index.html` no browser
- Navegue pelas categorias: **Imagem** e **Áudio**
- Clique no prompt para **copiar automaticamente**
- Use a busca para filtrar por palavra-chave
- Veja a **Curadoria do Time** — os melhores prompts destacados

### Para o admin (dono do banco)
- Clique no botão **⚿ Admin** no canto superior direito
- Digite o código de acesso
- A interface se torna editável: adicione, edite ou delete prompts diretamente no browser
- Tudo salva automaticamente — sem mexer em código, sem mandar arquivo para ninguém

## Tecnologia

- **100% frontend** — HTML, CSS e JavaScript puros. Zero dependências externas.
- **Persistência via localStorage** — todos os prompts adicionados pelo admin ficam salvos no browser.
- **Sem servidor** — abre direto no browser como arquivo HTML.
- **Responsivo** — funciona em desktop e mobile.
