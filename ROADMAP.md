# ROADMAP — MTRX LIBRARY

> O que já existe, o que está em andamento, e o que vem a seguir.

---

## ✅ Versão 1.0 — MVP (Pronto)

- [x] Site HTML completo, responsivo, tema dark
- [x] Identidade visual com cores da marca MTRX (laranja + preto + branco)
- [x] Categorias: Imagem e Áudio
- [x] Subcategorias: Ângulos, Ambientação, Sujeitos, Gêneros, Vocal, Mood
- [x] Prompts iniciais em todas as categorias
- [x] Botão de copiar prompt com um clique
- [x] Busca em tempo real
- [x] Filtros por categoria
- [x] Sistema de Curadoria (favoritos globais)
- [x] Modo Admin com senha oculta
- [x] CRUD completo no browser (add, edit, delete)
- [x] Persistência via localStorage
- [x] Documentação: README, guia admin, guia de prompts
- [x] Tudo em um único arquivo HTML (zero dependências)
