O QUE ESTA PRONTO
----------------------------------------

PRODUTO
- Biblioteca de prompts para imagem, audio, video e WhatsApp
- Busca em tempo real
- Filtros por categoria
- Tags clicaveis
- Curadoria do time
- Combos rapidos
- Copia de prompt com um clique

ADMIN
- Senha admin: mtrx2024
- Criar, editar e deletar prompts
- Marcar prompts como curadoria
- Criar combos
- Exportar JSON
- Exportar TXT
- Importar JSON
- Desfazer exclusao por 7 segundos

BANCO
- Supabase conectado
- Tabela principal: mtrx_library_state
- Backup local via localStorage
- SQL salvo em supabase.sql

DEPLOY
- Vercel publicado
- GitHub conectado
- Qualquer git push em main atualiza o site


MELHORIA DIRETA FEITA AGORA
----------------------------------------

A v1.1 iniciou a adaptacao para uma biblioteca maior:

- Layout mais responsivo para mobile, tablet e desktop
- Painel de metricas:
  - total de prompts
  - total de tags
  - total de combos
  - status do banco
- Resumo dinamico da busca e dos filtros
- Melhor comportamento de tags, combos e curadoria em telas pequenas
- Cards mais resistentes a textos longos
- Modais melhores no celular
- Suporte a prefers-reduced-motion


DOMINIO
----------------------------------------

O dominio principal NAO deve ser usado neste projeto.

mtrx.com.br ficara para a landing page principal.

Para a biblioteca, usar:
library.mtrx.com.br

DNS necessario:
Tipo: CNAME
Nome: library
Valor: 15726c4eab95e25c.vercel-dns-017.com.

Esse passo depende do Registro.br liberar/estabilizar a configuracao de DNS.


PROXIMOS PASSOS
----------------------------------------

1. Finalizar DNS do subdominio library.mtrx.com.br
2. Aumentar o banco de prompts por uso real
3. Padronizar tags por ferramenta
4. Criar historico de uso / prompts mais copiados
5. Criar login admin real com Supabase Auth
6. Preparar landing page futura no dominio mtrx.com.br
