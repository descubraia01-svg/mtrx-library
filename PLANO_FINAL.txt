PRÓXIMO PASSO IMEDIATO — DEPLOY VERCEL
----------------------------------------

1. No terminal, cole este comando:
   git push --set-upstream origin main

2. Acesse: vercel.com/new

3. Clique em "Continue with GitHub"

4. No dropdown de conta, selecione ou adicione:
   descubraia01-svg

5. Selecione o repositório: mtrx-library

6. Configurações:
   - Framework: Other (sem build)
   - Build Command: (deixar vazio)
   - Output Directory: ./ (padrão)

7. Clique Deploy

8. Em 30 segundos o site estará em:
   mtrx-library.vercel.app (ou URL gerada pela Vercel)

DEPOIS DISSO: qualquer git push atualiza o site automaticamente.


PRÓXIMOS PASSOS MÉDIO PRAZO — SUPABASE
----------------------------------------

Por que fazer:
- Hoje os dados ficam no localStorage do browser
- Se o browser limpar dados, tudo some
- Não sincroniza entre computadores ou pessoas da equipe

Como fazer (Claude faz tudo quando você tiver as credenciais):

1. Crie conta gratuita em: supabase.com
2. Crie novo projeto (aguarda ~2 minutos)
3. Em Settings > API copie:
   - Project URL  (ex: https://abc123.supabase.co)
   - anon public key  (começa com eyJhbGci...)
4. Me passe essas duas informações
5. Rode o arquivo supabase.sql no SQL Editor
6. Cole a anon public key no arquivo config.js
7. git push → site atualiza → dados agora são centralizados

O SQL da tabela de sincronização está pronto no arquivo supabase.sql.
