VALIDACOES NECESSARIAS ANTES DE ENTREGAR
1. Abrir https://library.mtrx.com.br
2. Testar Som, Imagem e Video no celular.
3. Entrar em Admin com senha mtrx2024.
4. Criar um prompt de teste.
5. Enviar uma imagem pequena como referencia.
6. Enviar um audio pequeno como referencia.
7. Confirmar que o player toca no card.
8. Exportar JSON.
9. Abrir Diagnostico.
10. Apagar o prompt de teste ou deixar como "Em revisao".

STORAGE SUPABASE
Para ativar upload publico direto no Supabase Storage, executar o arquivo:
supabase_storage.sql

Se o bucket ainda nao estiver ativo, o sistema usa fallback para arquivo leve direto no banco.

OBSERVACAO OPERACIONAL
Para videos grandes, usar URL publica como referencia.
Para arquivos pequenos, o botao Enviar arquivo resolve direto no cadastro.
